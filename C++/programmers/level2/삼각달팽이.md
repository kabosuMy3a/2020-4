#삼각달팽이  

아이디어는 1시간 정도 걸렸고 구현은 1시간 반이 걸렸다. 다른 사람의 풀이를 참조하지 않고 생각하고 생각대로 구현을 할 수 있어서 좋은 문제 풀이였던 것 같다.   

기본 아이디어는 n개의 list를 만들어서 각 각에 규칙에 맞게 정확한 위치에 원소를 넣는 것이다. n개의 list를 가지는 vector를 만들어 구현했다.    
```
vector<list<int>> lists ;  
for(int i = 0 ; i <= n ; i++){  
    list<int> l ;  
    lists.push_back(l);  
}  
```
